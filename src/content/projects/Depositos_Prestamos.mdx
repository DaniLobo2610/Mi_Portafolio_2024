## Contexto del proyecto

El proyecto es algo sencillo, está diseñado para que los empleados puedan gestionar sus préstamos y depósitos desde la consola. 
Mediante un menú interactivo, los empleados pueden realizar diversas operaciones de forma sencilla.
## Tecnologias utilizadas

### Frontend
- Java Console

### Backend

- Java Code "Visual_Code"
