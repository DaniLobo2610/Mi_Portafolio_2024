## Contexto del proyecto
Un programa en Java que implemente las Torres de Hanoi interactivo podría funcionar de la siguiente manera en la consola
- Input del Usuario: El programa solicita al usuario que ingrese el número de discos.
- Movimiento de Discos: Utiliza un algoritmo recursivo para resolver el problema, explicando paso a paso cada movimiento (por ejemplo, de qué varilla a cuál mover cada disco).
- Interacción en la Consola: Después de cada paso, el programa espera una confirmación o acción del jugador para continuar, permitiendo una experiencia interactiva y educativa.
