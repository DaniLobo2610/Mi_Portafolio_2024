## Contexto del proyecto

El proyecto "Expediente Médico" es una API REST, diseñada para gestionar integralmente información vital. Proporciona servicios de gestión 
y consulta de expedientes médicos en un entorno de salud. Implementada en un marco web, permite la interacción con una base de datos SQL Server 
mediante controladores que facilitan la creación, lectura, actualización y eliminación de registros médicos.
## Tecnologias utilizadas

### Backend

- APIs

### Base de datos

- SQLServer
