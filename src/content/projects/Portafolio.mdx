## Contexto del proyecto
Este proyecto consiste en el desarrollo de una página web de portafolio personal que presenta de 
manera profesional y visualmente atractiva las habilidades, tecnologías, y proyectos de un desarrollador.
La página web se construirá utilizando Astro, un moderno framework que permite optimizar el rendimiento y 
velocidad de carga
## Tecnologias utilizadas

### Frontend
- Tailwind CSS
- Astro "Principalmente"
- MDX "Principalmente"
### Backend
- Node Js
